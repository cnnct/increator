# ueditor**富文本标签**

#### ueditor**标签的属性 :**

> ueditor标签有2个属性分别为为id、name；
>
> **其中id为必填项,下面必填项加上了\*号**；
>
> > ** \*id ：** id属性
> >
> > **name ：** name属性

#### ueditor标签的引入方式 :

```
   <@ueditor id="editor" name="myEditor"/>
```
